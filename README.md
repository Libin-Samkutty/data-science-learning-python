# Data Science Learning — Python

A structured, hands-on Python data science curriculum covering NumPy, Pandas, and feature engineering through numbered lesson scripts.

## Structure

```
numpy/               # Lessons 01–16: NumPy fundamentals
pandas/              # Lessons 17–32: Pandas for data manipulation
feature_engineering/ # Lessons 33–34: Feature engineering & cleaning pipelines
dataset/             # Sample datasets used in exercises
```

## Topics Covered

**NumPy** — arrays, shapes, indexing/slicing, vectorization, broadcasting, aggregations, missing values, boolean masking, universal functions, sorting, performance, and debugging.

**Pandas** — Series/DataFrames, filtering, missing values, type conversion, groupby, merging/joining, time series, applying functions, large datasets, validation pipelines, method chaining, and EDA workflows.

**Feature Engineering** — feature creation techniques and end-to-end data cleaning pipelines.

## Setup

```bash
python -m venv venv
source venv/Scripts/activate  # Windows
pip install -r requirements.txt
```
